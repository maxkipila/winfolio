This chart represents the current new/sealed value of the set Spider-Man since May 2022. Very high accuracy
LEGO 30302 Spider-Man was a 45 piece Marvel Super Heroes set with 1 minifig released in 2014. It features Spider-Man on a glider with two flick missiles, movable side wing flaps. This set was available for sale in Toys R Us stores and free with a Daily Mail promotion.

The current value for a new and sealed Spider-Man is estimated at €8, with an average annual growth of approximately 5%, which is lower than other Ultimate Spider-Man sets. On secondary markets, the typical price range for the set MISB/NISB is between €6 and €8. On the open market, a used set will sell in the range of €4 and €5, depending on its condition.

About Marvel Super Heroes Ultimate Spider-Man

LEGO Ultimate Spider-Man is a subtheme of Marvel initially released in 2012 based on the animated television series based on the Spider-Man comics published by Marvel Comics.

Ultimate Spider-Man consists of 10 sets that ran from 2012 to 2015. As of today, 76005 Daily Bugle Showdown is the most valuable Ultimate Spider-Man set with an estimated new/factory sealed value of €211.
Summary
Marvel Super Heroes Ultimate Spider-Man Spider-Man set 30302 was released in March 2014, retailing for €4, but has since been retired. New, unopened Spider-Man sets are valued at €8, while used sets can be found for €4 in good condition.
